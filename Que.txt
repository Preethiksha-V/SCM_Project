E-Commerce Order Management System
Problem Statement:
Ensure customer orders are processed correctly with accurate inventory updates and payment
handling.
Tasks:
1. Develop order processing and inventory management logic
2. Write JUnit tests for order success/failure scenarios
3. Implement CI/CD pipeline with automated testing
o Set up version control using GitHub
o Configure CI pipeline (Jenkins / GitHub Actions)
o Automate build process using Apache Maven
o Integrate automated unit tests in pipeline
o Configure CD pipeline for deployment
4. Deploy services on Docker and Kubernetes
o Containerize application using Docker
o Push image to Docker Hub
o Create Kubernetes deployment and service YAML files
